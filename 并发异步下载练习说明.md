## 关于练习的问题：

- 小明最近学习了[Fluent Python]中的并发、异步等内容。其中包括并发/异步网络下载，对应的示例为"下载20多个国家国旗图像的程序代码"，里面有提到"测试服务器"，提示建议搭建一个，不如直接并发/异步在某网站下载会相当于拒绝服务DOS攻击,无法练习；
- 思考：是不是需要在WSL中部署一个？还是说有什么更高效可靠的方案？
- 
  使用方法

  # 1. 先安装 Pillow（用于生成图片）
  pip install Pillow

  # 2. 生成测试图片（会创建 flags/ 目录，里面放20张纯色PNG）
  python generate_flags.py

  # 3. 启动测试服务器
  python test_server.py

  服务器运行后，测试URL就是：
  http://localhost:8000/flags/CN.png
  http://localhost:8000/flags/US.png
  ...

  特点

  - 标准库实现，除了生成图片用了Pillow，服务器本身零依赖
  - 0.1秒延迟 让并发效果更明显（可修改 time.sleep(0.1) 调整）
  - 控制台实时打印 每个请求的时间戳，方便观察并发行为
